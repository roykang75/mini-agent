Agent Core Detail src/lib/agent.ts
async function* runAgent(userMessage: string): AsyncGenerator<AgentEvent>

New conversation entry point. Creates initial messages array and delegates to agentLoop.

async function* resumeAgent(session: Session, approved: boolean): AsyncGenerator<AgentEvent>

Resumes after HIL approval. If approved: executeTool for each pending call, push tool_results to messages. If rejected: push error tool_results, yield tool_rejected events. Then re-enters agentLoop.

async function* agentLoop(messages: MessageParam[]): AsyncGenerator<AgentEvent>

Core loop: while(true) call Anthropic API (claude-sonnet-4), yield text blocks as "message" events, detect tool_use blocks, createSession + yield "tool_approval_request", then return (pause). Exits on stop_reason "end_turn".

Output: SSE stream of AgentEvent objects

POST /chat/approve

src/app/chat/approve/route.ts

POST /chat/approve Approve or reject pending tools
Input: { sessionId: string, approved: boolean }
Validation: sessionId + approved required (400), session lookup (404)
Output: SSE stream of AgentEvent objects
Flow: validate -> getSession -> resumeAgent -> deleteSession -> Response

Tool Execution 3 Tools with Zod Validation
read_file
Read file contents and return as string
Input: { path: z.string() }
Impl: readFile(path, "utf-8")
Error: Returns error message string
File: src/lib/tools/read-file.ts
write_file
Write content to file with auto directory creation
Input: { path: z.string(), content: z.string() }
Impl: mkdir(recursive) + writeFile(utf-8)
Output: Success message with byte count
File: src/lib/tools/write-file.ts
run_command
Execute shell command with timeout and buffer limits
Input: { command: z.string() }
Impl: exec(command, { timeout: 30s, maxBuffer: 1MB })
Output: stdout + stderr combined
File: src/lib/tools/run-command.ts
